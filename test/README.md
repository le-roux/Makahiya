Instructions to install Criterion:
1 - clone the repository git@github.com:Snaipe/Criterion.git
2 - go in the directory
3 - $ mkdir build
4 - $ cd build/
5 - $ cmake ..
6 - $ cmake --build .
7 - $ make install
